# picoCTF-Writeups
This repository contains personal writeups for selected Capture The Flag (CTF) challenges hosted on picoCTF (play.picoctf.org). The focus is on documenting problem-solving methodology, tools used, and reasoning behind each solution.
Plaintext flags are not published.
## Categories
Use the links below to navigate to specific challenge categories:
- [Web Exploitation](https://github.com/msvignesh-25/picoCTF-Writeups/tree/main/WEB%20EXPLOITATION)
- [Cryptography](https://github.com/msvignesh-25/picoCTF-Writeups/tree/main/CRYPTOGRAPHY)
- [Reverse Engineering](https://github.com/msvignesh-25/picoCTF-Writeups/tree/main/REVERSE%20ENGINEERING)
- [Forensics](https://github.com/msvignesh-25/picoCTF-Writeups/tree/main/FORENSICS)
- [General Skills](https://github.com/msvignesh-25/picoCTF-Writeups/tree/main/GENERAL%20SKILLS)
- [Binary Exploitation](https://github.com/msvignesh-25/picoCTF-Writeups/tree/main/BINARY%20EXPLOITATION)
## Repository Structure
Challenges are organized by category. Each challenge typically contains a dedicated folder with a README explaining the solution process.
## How to Use This Repository
1. Select a challenge category from the links above  
2. Navigate to an individual challenge directory  
3. Read the `README.md` for step-by-step explanation  
4. Attempt challenges independently before reviewing solutions
